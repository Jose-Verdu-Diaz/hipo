Install

You will need Python and some modules to use HIPO. If you are not an experienced user you will find detailed instructions in the Environment section. Otherwise, feel free to jump to the Source Code section.

Environment

It is recommended to use Conda for installing Python and the required modules. From Conda documentation:

Conda is an open source package management system and environment management system that runs on Windows, macOS and Linux. Conda quickly installs, runs and updates packages and their dependencies. Conda easily creates, saves, loads and switches between environments on your local computer. It was created for Python programs, but it can package and distribute software for any language.

In other words, conda allows you to create “virtual environments” where you can install Python and modules. You can create as many environments as you like, and they are all independent between each other and your machine. This way you can have environments running different Python and module versions, and you can cleanly remove all modules from your machine by just removing the environment.

Conda is provided through Anaconda and Miniconda. Miniconda is a minimal version of Anaconda, including only the essentials for creating and managing Python environments. To install it, go here and select the option that suits your machine.

After installing installing conda, open a terminal. If you use Windows, you may open a terminal by pressing Windows Key + r, entering cmd in the input box and pressing Enter. After opening the terminal, enter conda -V you should see something similar to the following:

$ conda -V
conda 3.7.0

If the conda command is not found, you might not have conda in your PATH variable. Refer to the following article for detailed instructions.

Otherwise, you are ready to create an environment for HIPO! To do so, enter conda create -n hipo python.

$ conda create -n hipo python
Collecting package metadata (current_repodata.json): done
Solving environment: done

...

Proceed ([y]/n)?

When you are asked to proceed, enter y. After installing the environment, activate it with conda activate hipo. The terminal will update to display that the environment is active.

$ conda activate hipo
(hipo)$

You may deactivate the environment anytime by entering conda deactivate.

Now you are ready to install all required modules. Enter, one-by-one, the following commands. Yo will be asked to confirm after each one.

  • conda install pandas
  • conda install pillow
  • conda install tifffile
  • conda install matplotlib
  • conda install seaborn
  • conda install -c conda-forge seaborn-image
  • conda install tqdm
  • conda install -c conda-forge napari
  • conda install -c conda-forge tabulate

Your environment is now ready to run HIPO! In the next section you will find how to download the source code.

Source Code

You can download the source code in two ways:

  • (RECOMMENDED) Download the latest stable release
  • Clone or download the hipo repository, if you want to test the latest unreleased (and unstable) features.

Note that this documentation will always refer to the latest release. HIPO is currently in a very early, fast-changing development stage, so this documentation might not apply to older releases.

After obtaining the source code, uncompress it into a folder and and navigate to it with the terminal. If you are not familiar with navigating through a terminal, refer to the following article (Windows) or the following article (MacOs). When you are located in the HIPO folder, and with your environment active, enter python main.py. If you see the welcome output, congratulations, you finished setting up HIPO!
